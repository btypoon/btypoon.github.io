Blender-shortcut

Here list the useful blender-shortcut when working with 3D printing model

Change transformation pivot point

First press Tab To go into Edit Mode. Select the Face you want the origin pivot point to center on. Next, press Shift+S to bring up the Snap Menu. Choose Cursor to selected to snap the 3D cursor to the center of the selected face. Press Tab again to go back to Object Mode. In menu Object, select Set Origin, then Origin to 3D Cursor. The origin pivot point will then be snapped to the location of the 3D Cursor.

Alternatively, a shortcut can be set to the action Origin to 3D Cursor. Right click on the menu and select Assign Shortcut to give it a shortcut of your choice. Next time, after the 3D cursor has been snapped to a desired location, the shortcut can be pressed immediately to snap the origin pivot point to the 3D Cursor

If the built-in addon Interface: 3D Viewport Pie Menus was enabled, in Edit Mode, pressing Ctrl+Alt+X will bring up the Origin Menu. Choosing Origin To Selected will immediately snap the origin pivot point to the center of selected face

Locking Z-axis

Press Shift+Z to lock the z-axis in place when moving the selected object (G) in Object mode. Doing so will display the X and Y axis in red and green colour respectively indicating the movable plane.

Render mode in Real time

Press Z to bring up the shading menu and select Rendered to render the viewport in real time

Alternatively, you can press the button on the top right menu of the viewport

Aligning object

Enable the addon Align Tools. Press N to bring up the Transformation Menu on the right of the viewport. At the bottom is the Align Tools section. Hold Shift to select multiple object. For example, clicking Align Location: X will align the x position of all the previously selected objects to the x position of finial selected object

Search any function

Enable the addon Dynamic Context Menu. Press Spacebar to bring up the dynamic context menu and click search. Now you may type in any desired action and activate it with ease

Copy and pasting Actions to multiple objects

After selecting multiple object, make changes to the lastly selected object. Right click on any action you desired to apply on all selected object to bring up a menu. Click Copy Single to Selected will apply the action done on the last selected object to all other selected objects.

Toggle X-ray mode

Press Alt+Z to toggle X-ray mode for better seeing the model. In X-ray mode, you can select obstructed vertices.
